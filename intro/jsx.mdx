## About JSX

<List>
  <Appear>
    <ListItem>JSX is not HTML</ListItem>
    <ListItem>Don't underestimate nice looking, simple JSX</ListItem>
    <ListItem>JSX represents objects, called "react elements"</ListItem>
    <ListItem>It is transpiled to plain javascript at build time</ListItem>
  </Appear>
</List>

```notes
  Instead of artificially separating technologies by putting markup and logic in separate files, React separates concerns with loosely coupled units called “components” that contain both.
```
